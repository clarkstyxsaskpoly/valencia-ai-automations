# Valencia AI & Automations

Static marketing site for Valencia AI & Automations with a light premium enterprise theme.

## Files

- `index.html` contains the full page structure and copy.
- `styles.css` defines the brand system, responsive layout, and visual treatment.
- `script.js` handles mobile navigation, demo form feedback, and reveal-on-scroll animation.
- `favicon.svg` provides the geometric V brand mark.

## Preview

Open `index.html` in a browser, or serve the folder with any static file server.
